## Key findings (fetched March 2026)

### Market direction (CXToday, Forrester Q1 2025)
- CRM is shifting from system of record to system of action
- Bloated CRMs are losing to simpler, higher-adoption tools
- AI agents are becoming native (not add-ons) in Salesforce, Dynamics, Oracle
- #1 warning: bad data kills CRM ROI regardless of AI features

### Real estate pipeline patterns (Pipedrive real estate blog)
- Agents need MULTIPLE separate pipelines: buyers, sellers, investors, properties
- Buyer stages: lead capture → qualification → property matching → showing → offer → post-sale
- Seller stages: initial consult → listing setup → buyer inquiries → offer tracking → docs → move-in follow-up
- Key fields agents use daily: last contact date, budget, timeline, lead source, communication history
- #1 automation agents use: follow-up reminders + drip emails

### Pipeline stage philosophy (Pipedrive, Zoho, HubSpot)
- No vendor hardcodes real estate stages — all are user-defined
- Stage customization is table stakes
- Bottleneck visibility per stage is a core UI pattern
- Conversion rate per stage is expected in analytics

### What differentiates winning CRMs for agents
- Mobile-first (agents are in the field)
- Fast data entry (audio notes, quick add)
- Follow-up reminders with overdue detection
- Deal-linked documents and contracts
- Communication history visible on every deal
